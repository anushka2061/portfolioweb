Aawart Don CTF

In this project, I participated in the **Aawart Don CTF** challenge, a thrilling challenge that tested my knowledge and skills in network security and web exploitation. The challenge had multiple stages, each requiring a different approach to solve.

Challenge Overview

The Aawart Don CTF was designed to challenge participants on various aspects of web and network security. The task involved identifying vulnerabilities in both the server and the web application, leading to the final capture of the flag.

Steps Taken

1. **Initial Reconnaissance**: I started by scanning the target machine to find open ports and services using **Nmap**.
2. **Service Enumeration**: After identifying open ports, I explored the web service on port 80 using **Dirbuster** to look for hidden files and directories.
3. **Exploiting Web Vulnerability**: Upon finding a vulnerability in the web application, I used **SQL injection** to access the database and retrieve sensitive information.
4. **Final Capture**: After exploiting a privilege escalation vulnerability on the server, I successfully accessed the flag and submitted the solution for a point gain.

Challenges Faced

- The first challenge was identifying a **hidden file directory** on the server that was crucial for discovering the vulnerability.
- **SQL injection** was more complex than initially expected, requiring multiple attempts to bypass filtering mechanisms.
- The final privilege escalation involved multiple steps of reverse-engineering, making it difficult to navigate.

Solution

After gathering all the information, I was able to craft a SQL injection payload that bypassed the web application's input validation. This allowed me to access the internal database, which held critical information about the server's file system and users. After escalating my privileges, I gained access to the root directory and successfully captured the flag.

Key Learnings

- Gaining expertise in **SQL injection** and bypassing web application filters.
- **Privilege escalation** techniques and their importance in gaining full access.
- Using tools like **Nmap**, **Dirbuster**, and **Burp Suite** effectively in real-world environments.
- The importance of careful and systematic **reconnaissance** and **exploitation** in CTF challenges.
