Hacker CTF

In this project, I participated in the **Hacker CTF**, which focused on real-world scenarios related to web security, network attacks, and system exploitation. The challenge involved multiple phases, each requiring a unique approach and various tools to complete.

Challenge Overview

The **Hacker CTF** challenge provided a series of web and network security problems that needed to be exploited. Each phase required using tools like **Burp Suite**, **Nmap**, and **Metasploit**, alongside custom scripting, to discover vulnerabilities, escalate privileges, and ultimately capture the flags.

Steps Taken

1. **Reconnaissance**: I began by scanning the machine using **Nmap** to find open ports and services. This helped me identify the vulnerable web application running on port 80.
2. **Web Exploitation**: I used **Burp Suite** to identify and exploit SQL injection vulnerabilities in the web application, which allowed me to extract sensitive database information.
3. **Privilege Escalation**: After gaining initial access, I attempted privilege escalation on the system using **Metasploit** and found an unpatched vulnerability in the kernel, which gave me root access.
4. **Flag Capture**: I successfully captured the flag after exploiting the system, following the correct sequence of operations and finding hidden files with the flag.

Challenges Faced

- **Reconnaissance** was more difficult than expected due to the host being protected by basic security measures, including an aggressive firewall.
- **Web vulnerabilities** were well-hidden behind obfuscation, requiring a lot of trial and error to bypass the application security.
- **Privilege escalation** was another challenge, as the system required multiple stages of exploitation, which required careful research.

Solution

After identifying a **SQL injection** vulnerability, I was able to dump the database and find usernames and passwords. I leveraged these credentials to escalate privileges. The final step involved exploiting an unpatched kernel vulnerability, allowing me to access the root system and capture the flag.

Key Learnings

- The importance of **reconnaissance** and gathering detailed information before making any attacks.
- Using **Burp Suite** for identifying and exploiting web vulnerabilities, especially **SQL injection**.
- Privilege escalation techniques, including exploiting **kernel vulnerabilities** and other misconfigurations.
- How **systematic exploitation** and **cautious trial and error** can lead to success in a real-world scenario.
